📁 ARQUIVOS DO PROJETO:

HTML (Páginas):
├── index.html ............... Página inicial com banner promocional
├── camisas.html ............. Catálogo completo de produtos
├── exclusivo.html ........... Camisetas raras (colecionador)
├── carrinho.html ............ Carrinho de compras
├── checkout.html ............ Finalização do pedido
├── login.html ............... Login de usuário
├── cadastro.html ............ Cadastro de novo usuário
└── perfil.html .............. Perfil do usuário

CSS:
└── css/style.css ............ Todos os estilos personalizados

JAVASCRIPT:
├── js/main.js ............... Funções principais, header, footer
├── js/products.js ........... Dados dos produtos
├── js/cart.js ............... Lógica do carrinho de compras
└── js/auth.js ............... Sistema de autenticação

🚀 COMO EXECUTAR:

OPÇÃO 1 - Abrir diretamente:
   • Dê duplo clique no arquivo "index.html"
   • O site abrirá no seu navegador padrão

OPÇÃO 2 - Servidor local (recomendado):
   
   Python 3:
   python -m http.server 8000
   
   Python 2:
   python -m SimpleHTTPServer 8000
   
   Node.js:
   npx http-server -p 8000
   
   Depois acesse: http://localhost:8000

💡 DICAS:

1. Use o modo escuro! Clique no ícone de lua/sol no header

2. Experimente adicionar produtos ao carrinho e ver o contador 
   atualizar

3. Os dados ficam salvos mesmo se você fechar o navegador
   (localStorage)

4. Para "limpar" tudo: abra o Console do navegador (F12) e digite:
   localStorage.clear()

═══════════════════════════════════════════════════════════════════

🎯 FLUXO DE USO COMPLETO:

1. Abra index.html
2. Clique em "COMPRAR AGORA" no banner
   → Produto adicionado ao carrinho
3. Navegue para "Camisas" ou "Exclusivo"
4. Adicione mais produtos
5. Clique no ícone do carrinho (verá o contador)
6. Ajuste quantidades
7. Clique em "Finalizar Pedido"
8. Preencha os dados (já vem preenchido)
9. Clique em "Finalizar Compra"
10. Sucesso! Será redirecionado
